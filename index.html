Prework Study Guide

HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user.
  • URL stands for Uniform Resource Locator.
  • Declarations are not case sensitive, but HTML elements are case sensitive.
  • Metadata titles are important for SEO or Search Engine Optimization.
  • Semantic HTML Elements are elements which describe their content and their job on the webpage.
  • Elements nested within other elements are called Child Elements, and the enveloping element would be the Parent Element.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • There are two types of boxes in CSS: Block and Inline boxes.
  • Use the wildcard symbol (*) to apply a rule to all elements visual on a page
  • If applying the same rule to multiple selectors, use a comma to simplify
  • Don't Repeat Yourself (DRY): a coding principle encouraging reuse and share coding whenever possible in order to reduce the number of lines in your code

Git

  • git status displays our current branch
  • git checkout branch-name switches to that branch
  • git checkout -b branch-name creates a new branch and switches to it
  • git pull origin main will check if your current branch is in sync
